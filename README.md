## Характеристика проекта

В ходе работы были реализованы:

- веб-приложение для управления заметками;
- сохранение заметок в `localStorage`;
- поддержка офлайн-режима через `Service Worker`;
- кэширование статических ресурсов приложения;
- подключение `Web App Manifest`;
- установка приложения как PWA;
- поддержка иконок и meta-тегов для мобильных платформ;
- реализация архитектуры `App Shell`;
- динамическая загрузка контента страниц;
- запуск приложения по `HTTPS`;
- обмен событиями в реальном времени через `WebSocket` (`Socket.IO`);
- push-уведомления при добавлении новых заметок;
- возможность подписки и отписки от push-уведомлений;
- добавление заметок с напоминанием;
- серверное планирование напоминаний через `setTimeout`;
- возможность отложить напоминание на 5 минут;
- страница «О приложении»;
- удаление заметок из списка.

## Функциональность

### Заметки

- просмотр списка заметок;
- добавление обычных заметок;
- удаление заметок;
- сохранение данных в `localStorage`;
- работа приложения при отсутствии сети;
- отображение заметок после перезагрузки страницы.

### PWA и офлайн-режим

- регистрация `Service Worker`;
- кэширование HTML, JavaScript, manifest и иконок;
- загрузка приложения из кэша;
- поддержка `Web App Manifest`;
- установка приложения на устройство как PWA;
- открытие приложения в отдельном окне после установки.

### App Shell

- быстрый показ каркаса интерфейса;
- отдельная загрузка динамического содержимого;
- страница «Главная»;
- страница «О приложении»;
- кэширование статических ресурсов и динамического контента.

### Уведомления и WebSocket

- подключение клиента к серверу через `Socket.IO`;
- отправка события при создании новой заметки;
- получение события другими вкладками в реальном времени;
- отображение всплывающего уведомления о новой заметке;
- push-уведомления через `Push API` и `web-push`;
- включение и отключение уведомлений через интерфейс.

### Напоминания

- создание заметки с датой и временем напоминания;
- хранение напоминания в поле `reminder`;
- планирование уведомления на стороне сервера;
- отправка push-уведомления в нужное время;
- кнопка «Отложить на 5 минут» в уведомлении;
- повторная отправка напоминания после откладывания.

## Что было реализовано

## Практическая работа №13

В проект добавлено базовое приложение для заметок с офлайн-поддержкой.

Реализовано:

- форма добавления заметок;
- список заметок;
- сохранение заметок в `localStorage`;
- регистрация `Service Worker`;
- кэширование основных файлов приложения;
- загрузка страницы из кэша при отсутствии сети.

---

## Практическая работа №14

Проект расширен поддержкой `Web App Manifest`.

Реализовано:

- файл `manifest.json`;
- набор иконок разных размеров;
- подключение manifest в `index.html`;
- meta-теги для мобильных платформ;
- кэширование `manifest.json` и иконок в `Service Worker`;
- возможность установки приложения как PWA.

---

## Практическая работа №15

В проект добавлены `HTTPS` и архитектура `App Shell`.

Реализовано:

- запуск приложения по защищённому протоколу `HTTPS`;
- генерация локальных сертификатов;
- каркас приложения с навигацией;
- динамическая подгрузка страниц `home` и `about`;
- стратегия `Cache First` для статических ресурсов;
- стратегия `Network First` для динамического контента;
- офлайн-доступ к страницам приложения.

---

## Практическая работа №16

В проект добавлены `WebSocket` и push-уведомления.

Реализовано:

- сервер на `Node.js` с использованием `Express`;
- подключение `Socket.IO`;
- отправка события `newTask` при добавлении заметки;
- получение события `taskAdded` в других вкладках;
- отображение всплывающих уведомлений в интерфейсе;
- подписка на push-уведомления;
- отписка от push-уведомлений;
- отправка push через `web-push`;
- маршруты `/subscribe` и `/unsubscribe`.

---

## Практическая работа №17

В проект добавлена детализация push-уведомлений с напоминаниями.

Реализовано:

- форма добавления заметки с напоминанием;
- поле выбора даты и времени;
- новая структура заметки: `id`, `text`, `reminder`, `createdAt`;
- серверное хранение активных напоминаний;
- планирование уведомлений через `setTimeout`;
- push-уведомление в заданный момент времени;
- обработка `notificationclick` в `Service Worker`;
- кнопка «Отложить на 5 минут»;
- маршрут `/snooze` для переноса напоминания.
