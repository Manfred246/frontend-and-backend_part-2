# Практики 13–17: PWA заметки с push-напоминаниями

Этот проект объединяет функциональность практических занятий 13–17:
- офлайн-режим и кэширование через Service Worker;
- Web App Manifest и установка как PWA;
- App Shell и загрузка контента через `fetch`;
- WebSocket через Socket.IO;
- push-уведомления;
- напоминания с кнопкой **«Отложить на 5 минут»**.

## Что реализовано

- обычные заметки;
- заметки с датой и временем напоминания;
- сохранение данных в `localStorage`;
- получение событий в реальном времени между вкладками;
- push-уведомления даже при неактивной вкладке;
- обработка действия уведомления в Service Worker.

## Запуск

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Запустить сервер:
   ```bash
   npm start
   ```

3. Открыть:
   ```text
   http://localhost:3001
   ```

4. Нажать **«Включить уведомления»** и разрешить уведомления в браузере.
